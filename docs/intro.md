# Introduction to ROS 2

In this lesson, we will learn the basics of **ROS 2 (Robot Operating System)** and how it serves as the robotic nervous system for humanoid robots.

## Topics Covered

- What is ROS 2?
- ROS 2 Nodes, Topics, and Services
- Setting up your ROS 2 workspace
- Communication between Python agents and ROS 2 using `rclpy`
- Understanding URDF (Unified Robot Description Format)

## Learning Outcomes

By the end of this lesson, students will be able to:

1. Explain the role of ROS 2 in robot control.
2. Create and run simple ROS 2 nodes in Python.
3. Simulate a basic humanoid robot using URDF.
